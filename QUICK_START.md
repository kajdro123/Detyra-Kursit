# Quick Start Guide - Dental Clinic Management System

## TL;DR - Get Running in 5 Minutes

### Step 1: Start Backend
```bash
cd backend
npm install              # if not already done
npm run dev             # starts on http://localhost:7000
```

### Step 2: Seed Demo Data (in new terminal)
```bash
cd backend
npm run seed            # creates test user and sample data
```

### Step 3: Start Frontend (in another new terminal)
```bash
cd frontend
npm install              # if not already done
npm run dev             # starts on http://localhost:5173
```

### Step 4: Login
- Open http://localhost:5173 in your browser
- Login with: 
  - **Email**: clinic@example.com
  - **Password**: password123

That's it! You're ready to use the Dental Clinic Management System.
